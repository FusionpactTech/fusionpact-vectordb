# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in FusionPact, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Email security concerns via the FusionPact website contact form at https://fusionpact.com
3. Include: description, reproduction steps, and impact assessment

We will acknowledge within 48 hours and provide a fix timeline.

## Supported Versions

| Version | Supported |
|---|---|
| 2.x | ✅ |
| 1.x | Security fixes only |
